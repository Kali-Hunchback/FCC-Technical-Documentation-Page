Introduction

C# syntax is highly expressive, yet it is also simple and easy to learn. The curly brace syntax of C# will be instantly recognizable to anyone familiar with C, C++, or Java. Developers who know any of these languages are typically able to begin to work productively in C# within a short time. C# syntax simplifies many of the complexities of C++ and provides powerful features such as nullable types, enumerations, delegates, lambda expressions, and direct memory access. C# supports generic methods and types, which provide increased type safety and performance, and iterators, which enable implementers of collection classes to define custom iteration behaviors that are simple to use by client code. Language-Integrated Query (LINQ) expressions make the strongly typed query a first-class language construct.

As an object-oriented language, C# supports the concepts of encapsulation, inheritance, and polymorphism. All variables and methods, including the Main method, the application's entry point, are encapsulated within class definitions. A class may inherit directly from one parent class, but it may implement any number of interfaces. Methods that override virtual methods in a parent class require the override keyword as a way to avoid accidental redefinition. In C#, a struct is like a lightweight class; it is a stack-allocated type that can implement interfaces but does not support inheritance.

Installing C#

The first and simplest option if you are a Windows or MacOS user is to go download Visual Studio for your Operating System here
Once you've done that, run the installer and choose a suitable place for the installation. At some point, you'll be asked which 'workloads' you would like to install. For the code examples below we'll only need .Net CORE desktop development. Once the installer is complete, we're ready to go!

For Linux users or individuals that would like to utilize VS Code instead of Visual Studio you'll simply need to download the copy of VSCode relevant to your OS here
and run the associated installer.
Additionally in order to compile the code, it is necessary to download the most recent .Net CORE SDK (available here) and the C# extension from the VS Code Market Place (fourth tab down on the left-most bar). It is recommended to restart your computer before continuing to enable the instantiation of all items necessary on the path. and VS Code is good to go with C#

C# Code Examples

Opening C#

If you were a Visual Studio User, simply double click the icon and select "create new project" and select the "Console App (.Net CORE) option", name it and choose a place to store the project and the viewer will open for you

If you were a VS Code User open your command line interface (CLI), navigate to the file desired for creation and enter 

dotnet new console

A notification will appear, click yes and the viewer will finish loading dependencies and assets. When you want to run your code simply enter 

dotnet run

Inside of the containing file.

C# Hello World

Once you've selected the project, you may realize that the Hello, World program is loaded by default, it should appear as below: 

using System;
						
namespace HelloWorld
{
 class Program
 {
	 static void Main(string[] args)
	 {
		 Console.WriteLine("Hello World!");
	 }
 }
}

There is a few things to note about this program, the first is the using System in the top line of the editor. This is C#'s way of Input and Output of functions. In this case, we are calling functions stored in the System package, which contains input and output functionality for the console.
The second is in the next line and is namespace HelloWorld this is the name of the project which you entered and represent the sum of your code all the code inside of the project and acts as a way to reference the program it's self.
Next we have 

static void Main(string[] args)

The Main functions represent the starting point of our program and will always begin from this point whenever the project is compiled or run. It accepts an array of strings as arguments.

Finally we have 
Console.WriteLine("Hello, World")

This line of code accesses the Console and then calls for the

WriteLine(string)

Method which accepts a string parameter which is text. Whatever is placed inside of the parentheses and quotes will be output onto the console screen (Try changing the text inside of the quotes in your editor now!) and click the run button in the top toolbar or for VS Code or linux users enter

dotnet run

Inside of the file container in your CLI
